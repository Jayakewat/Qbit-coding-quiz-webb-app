# Qbit - Coding Quiz web Application

This is a full-stack coding quiz application developed using the MERN stack (MongoDB, Express.js, React, Node.js). This platform allows users to securely sign up and log in, attempt multiple quizzes across various programming languages, and view their quiz results after completion.
The application features user authentication, dynamic quiz management, score calculation, and a responsive UI for a smooth user experience. It is designed to help users test and improve their programming knowledge through interactive quizzes.

##  Live Demo:

 [Click here to view the app](https://qbit-coding-quiz-webb-app.vercel.app/)

## Features

- User authentication (Sign Up / Login)
- Secure authentication using JWT
- Multiple quizzes on different programming languages
- Real-time score calculation
- Quiz result summary
- Responsive and user-friendly interface
